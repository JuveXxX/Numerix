C'est le seul fichier HTML du projet. Son rôle est purement structurel : déclarer les éléments du DOM avec leurs id et aria, charger les polices et les feuilles de style, et inclure les scripts en fin de <body> dans le bon ordre.Il ne contient aucun onclick inline — tous les événements sont branchés depuis game.js. La balise <main class="nx-root"> contient 4 blocs : l'en-tête magazine, les badges de contraintes, le stepper éditorial, et la carte de jeu. Les id comme nx-input, stat-coups, nx-feedback sont les points de contact exclusifs que ui.js utilise pour écrire dans le DOM.
