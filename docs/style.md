14 sections clairement séparées par des commentaires. Aucune dépendance externe (Bootstrap a été retiré). Tout le thème est piloté par 7 variables CSS en tête de fichier (--nx-ink, --nx-paper, --nx-ground, etc.), ce qui permet de changer l'ambiance en modifiant un seul bloc.
Points notables : la typographie bipolaire (Playfair Display pour les grands chiffres et titres, DM Mono pour tous les labels et données), les 3 états de couleur sur les stats (normal → .warn orange → .danger rouge), et la correction de la media query mobile qui était syntaxiquement cassée dans la version originale (@media screen(...) → @media screen and (...)).
